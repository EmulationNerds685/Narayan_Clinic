# Narayan Heart & Maternity Centre Patient Portal

A healthcare appointment booking and patient education platform developed for Narayan Heart & Maternity Centre. The application enables patients to schedule appointments, explore healthcare services, watch educational content, and interact with the clinic through a modern, mobile-friendly interface.

## Live Demo

🔗 **Website:** https://narayanheartandmaternitycentre.com/

## Features

* OTP-based appointment booking workflow
* Doctor and healthcare service information
* Patient education video integration
* Feedback and testimonial system
* Contact and enquiry forms
* Mobile-responsive user experience
* SEO-optimized pages for local healthcare discovery
* Modern UI built with Material UI and Tailwind CSS
* Smooth animations using Framer Motion
* Performance-focused React architecture

## Tech Stack

### Frontend

* React
* Vite
* React Router
* Tailwind CSS
* Material UI (MUI)
* Framer Motion
* Axios

### Integrations

* REST APIs
* YouTube Data API
* OTP Verification System
